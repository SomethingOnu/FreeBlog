独断と偏見で紹介するソフトウェア

ブログの内容が思い付かないので、おぬ氏が利用しているWindowsのソフトウェアを(無料・有料問わず)テキトーに紹介します。言わずともフリーソフト酷使マンなのでほぼ無料ですが。

ビッグな人間へ成った何時には恩を籠めて寄付でもしたい。

一覧

ブラウザー・プレイヤー

MDIE

ファイルブラウザー
  • 価格 : 無料
  • 入手 : cres.s28.xrea.com

Windows標準のエクスプローラーにタブ機能が付属したようなファイル管理ソフト。機能も充実しており、特に説明書を読まずとも直感で操作できるという理由で何となく使っていた(新しい環境では諸事情により使っていない)が2008年から更新がないため、そろそろ主も新しい類似品へ移行しようか迷っている。

7-Zip

圧縮ファイルブラウザー
  • 価格 : 無料
  • 入手 : sevenzip.osdn.jp

大抵のパソコンにはWinZipかWinRARが入っていると思うが、選ばれたのは7-Zipでした。圧縮・解凍ウィンドウも使いやすければブラウザーも基本的な操作が行えるので良し。何より皆大好き超圧縮7Zファイルが扱えるので重宝している。

BunBackup

バックアップ
  • 価格 : 無料
  • 入手 : nagatsuki.la.coocan.jp

ファイル元とファイル先を設定してボタンを押せばバックアップ！むしろバックアップソフトにそれ以外を求めることはないが、自動バックアップや世代管理などプレッパー気質の自分にとって便利な機能が揃っていたので採用。バグもないので安心安全。

お～瑠璃ね～む

ファイル・フォルダーリネーム
  • 価格 : 無料
  • 入手 : beefway.sakura.ne.jp

ファイルやフォルダーの名前を一括で変更するソフトウェア。文字列の追加・削除だったり置換や連番など、名前と拡張子については自由な編集が可能。UnDo(元に戻す)操作やタイムスタンプ編集機能も地味に便利。

Firefox

インターネットブラウザー
  • 価格 : 無料
  • 入手 : www.mozilla.org

どう考えてもGoogle Chromeを入れるべきだが、反抗期なのでFirefoxを使っている。これと言った紹介はないが、設定で履歴・クッキーを常に削除する機能があるので好き。プラグインもそれなりに揃っているので満足。関係ないけどMozillaのリファレンス(MDN Web Doc)って便利だよね。

Media Player Classic

動画プレイヤー
  • 価格 : 無料
  • 入手 : ja.osdn.net

Windowsのデフォルトアプリがうんｋ動画再生プレイヤーに色々と機能を求めるならコレ一択。MP4ファイルからDVDメディアまでフォーマットを気にすることなく動画を視聴することができれば、困ったときは設定を弄れば全て解決する。映像音声同期やフィルターなど、痒いところにも手が届く。

AIMP

音楽プレイヤー
  • 価格 : 無料
  • 入手 : www.aimp.ru

豊富なスキンが売りの多機能オーディオプレイヤーことAIMP君です。多くのフォーマットに対応しており、シンプルなボタンには必要十分の機能が詰まっている。再生機能・編集機能を兼ね備えた便利なソフトウェアだが、オンラインコンテンツが充実している現代ではメディア再生ソフトの需要が減っているという悲しい現実。そのうち大容量HDDみたいな怪しいポジションになるのかな…。

System Explorer

タスクマネージャー
  • 価格 : 無料
  • 入手 : systemexplorer.net

Windows標準のタスクマネージャーも有能だが、ウィンドウからファイル元を特定してくれる機能やシステム情報にアクセスできる機能を持つコレは素晴らしい。他にもネットワークの監視だったりプロセスの解析、ソフトウェアのアンインストールが行えるなど、上手に使えばコンピューターの健康カルテとして利用もできる。

制作・編集・変換

CatMemoNote

文章制作
  • 価格 : 無料
  • 入手 : www.cc9.ne.jp

とにかく装飾や機能設定の自由度が高いテキスト(TXTファイル)エディター。テキスト表示・タブ表示・定型文・キーボードショットカットなどは高級ソフトウェア相応なので、小説執筆ソフトとして愛用している。操作方法も簡単でレジストリーを使用しないポータブル可能な設計。縦書き機能や折り返し文字数指定に関しては神。なので逆にスマホから書くのを躊躇うようになってしまった…。

TeraPad

文章編集
  • 価格 : 無料
  • 入手 : tera-net.com

テキストエディター界では派閥による対立が激しい中、個人的に勧めるのは基本的な機能が完備されているコレ。シンプルながらも不満ポイントもないので、使い続けるうちに常用ソフトとなった。ぶっちゃけ主は2割ぐらいの機能しか使っていない気がする。注意点として、ユニコードを扱うファイルを編集する場合は然るべき設定をしましょう。私はやらかした。

Visual Studio Code

コード編集
  • 価格 : 無料
  • 入手 : azure.microsoft.com

先程のCatMemoNoteやTeraPadでもコードの編集はできるが、やはり本格的な作業をするならMicrosoft様の強々ソフトウェアに頼るしかない。強化された機能や充実したプラグインなどは数多くの技術者が愛用しているだろう。特にGitHubなどのコード管理サービスと連携すればチーム作業効率は最大値まで上がる。金と集合知の力って凄い。

GIMP

画像制作
  • 価格 : 無料
  • 入手 : www.gimp.org

プロが使うような画像処理ソフトと同等の機能を持つ無料のソフトウェア。最初こそ慣れは必要だが、何れは自分が表現したいモノを忠実に作り上げることができる。レイヤー機能、ブラシ、エフェクト、ベクター編集、スクリプトで自動処理…等々、編集者にとっては強い味方だろう。

CLIP STUDIO PAINT

イラスト制作
  • 価格 : DEBUT版 : 非売品 / PRO版 : 5,000円～ / EX版 : 23,000円～
  • 入手 : www.clipstudio.net

イラスト業界の覇者といえば、そう、クリスタだ。これ一つで簡単なイラストからアニメーションまで全てが描ける。豊富なペンとツールを駆使して、デッサン人形を参考に人物を描くことも、ブラシで記号的な表現を描くことも、イラストの道を歩む者であればコレを買って損はない。ちなみに金銭的余裕がなければMediBang Paintもオススメ。

OBS Studio

動画録画
  • 価格 : 無料
  • 入手 : obsproject.com

ディスプレイやウィンドウを録画するソフトウェアは幾つか存在するが、OBSはそれらと一線を画している。録画用の枠には複数のメディア、テキスト、オブジェクトを配置することができるため、雛型を持つ動画やリアルタイム配信では特に制作の効率化を図ってくれるだろう。なぜ無料で使えるのか分からないレベル。

AviUtl

動画制作
  • 価格 : 無料
  • 入手 : spring-fragrance.mints.ne.jp

動画編集者であれば一度は聞いたことがある、動画編集ソフトの分厚い鉄板。プラグインを追加することで様々な機能が使えるようになり、使いこなせばプロレベルの映像も作れたり、というか実際に作られている。初心者には入門が難しいことで有名だが、少し使えばマジで慣れるから！ゆっくり実況動画とか作りたいでしょ？だったら強制的に使わないとね？

Audacity

音声編集
  • 価格 : 無料
  • 入手 : www.fosshub.com

主は音声データをほとんど扱わないが、とりあえずコレを使えば困ることはないと断言できる。波形を見ながら音声を切り貼りしたり、エフェクトを付加したり、余程の要求がなければAudacity一つで完結するだろう。VSTプラグインも使えるらしいので、高機能なサウンド編集ソフトだと思う。

Blender

ＣＧ制作
  • 価格 : 無料
  • 入手 : www.blender.org

AviUtlよりも入門の敷居が高い(と言われる)３ＤＣＧ編集ソフト。まあＣＧ自体が難しいから仕方ないけど…しかしBlenderは数少ないＣＧソフトの中でも無料で使える逸品なのだ。しかも高機能・多機能でハイクオリティーのモデルが作れる。モデルどころか頑張れば映画一本も作れる。何だコイツ。

Caesium Image Compressor

画像変換
  • 価格 : 無料
  • 入手 : saerasoft.com

ラスター画像(PNGとかJPGとか)を一括でリサイズ・圧縮・最適化できるソフトウェア。例えばカメラで撮った写真を閲覧用にリサイズしたり、ブログに貼り付ける画像を圧縮したり。画像を変換する、その動作はコレ一つで足りる。

XMedia Recode

動画変換
  • 価格 : 無料
  • 入手 : www.xmedia-recode.de

画像変換に続き、こちらは動画変換。内容も画像から動画に置き換わっただけの説明だが、幾つもある動画フォーマットに全て対応できるのが強い特徴。拡張子に合わせて細かいプロパティーが設定できたり、解像度の変更やクリッピング、タグの編集も行えるという充実っぷり。

システム・ユーティリティー

CCleaner

システムクリーナー
  • 価格 : 無料
  • 入手 : www.ccleaner.com

界隈によっては賛否両論だが、やはり有ると便利なソフトウェアの一つ。綺麗好きの主が手放せるわけもなく、定期的に詳細クリーンとドライブワイパーを実行している。あくまでもログを消したいだけなのでレジストリー系は絶対に触らぬ。触ると何が起こるのか、経験者は背中で語る。

Desk Deck

ランチャー・その他
  • 価格 : 無料
  • 入手 : github.com

オススメというか自分用に作ったソフトウェア。作業時に使う機能：BGMの再生やソフトウェアの起動、CPU使用率・現在時刻の確認を一つのウィンドウで済ませたいという理由で作られたのがコレ。HSP製なので動作は不安定気味だが、クールなデザインと外見は皆に自慢したい。

Nube Tool Box

雑多・その他
  • 価格 : 無料
  • 入手 : github.com

同じく自作したソフトウェア。こちらは頻繁に使う簡単な機能を纏めた「ツールボックス」を目指して作ったもの。独特なUIと操作性だが、自分の成果物なので愛用している。やっぱりエゴ満載のソフトって使いやすいよね。
